# Settings

## Inline Titles

`Appearance` -> `Show Inline Title` (turn off)  

## Templates

`Core Plugins` -> `Templates` (turn on) -> ⚙️
- set folder location to \_\_Templates  
- set date/time structure

## Workspaces

`Core Plugins` -> `Workspaces` (turn on)
